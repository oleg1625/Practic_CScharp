Этот репозиторий содержит мою уникальную учебную работу. Код публикуется в образовательных целях и для демонстрации истории его разработки.

**Любое прямое копирование моей работы другими студентами для сдачи в качестве своей собственной является нарушением правил академической честности.**

Мои одногруппники имеют доступ к этому коду для ознакомления и обучения, но не для плагиата.
